# Ensemble Learning in Machine Learning

This repository contains implementations of Ensemble Learning techniques using Python and Scikit-learn.

## Project Overview

The notebook demonstrates how multiple machine learning models can be combined to improve prediction performance and model accuracy.

## Concepts Covered

- Ensemble Learning
- Voting Classifier
- Bagging
- Random Forest
- Boosting
- Stacking
- Model Training
- Model Evaluation
- Classification Algorithms

## Algorithms Used

### Base Models
- Logistic Regression
- Decision Tree Classifier
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

### Ensemble Techniques
- Voting Classifier
- Bagging Classifier
- Random Forest Classifier
- AdaBoost Classifier
- Gradient Boosting
- Stacking Classifier

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Workflow

1. Load Dataset
2. Data Preprocessing
3. Train-Test Split
4. Train Base Models
5. Apply Ensemble Techniques
6. Compare Model Performance
7. Evaluate Accuracy Scores
